# Internationalization (i18n)

The React SDK supports multiple languages out of the box with Vietnamese and English. You can also provide custom locales.

## Built-in Locales

```typescript
import { viLocale, enLocale } from "ermis-ekyc-react";
```

| Locale     | Language                | Default? |
| ---------- | ----------------------- | -------- |
| `viLocale` | Vietnamese (Tiếng Việt) | ✅ Yes   |
| `enLocale` | English                 | ❌ No    |

## Usage

```tsx
import { EkycMeetingProvider, enLocale } from 'ermis-ekyc-react';

// English UI
<EkycMeetingProvider locale={enLocale} {...otherProps}>
  {children}
</EkycMeetingProvider>

// Vietnamese (default, no need to specify)
<EkycMeetingProvider {...otherProps}>
  {children}
</EkycMeetingProvider>
```

## Custom Locale

Create your own locale by implementing the `EkycLocale` interface:

```typescript
import type { EkycLocale } from "ermis-ekyc-react";

const jaLocale: EkycLocale = {
  preview: {
    title: "デバイスチェック",
    description: "セッションに参加する前にカメラとマイクをテストしてください",
    joinButton: "セッションに参加",
    joining: "接続中...",
    cameraLabel: "カメラ",
    micLabel: "マイク",
    cameraOff: "カメラがオフです",
  },
  room: {
    connecting: "会議室に接続中...",
    hostLabel: "審査員 (HOST)",
    guestLabel: "顧客 (GUEST)",
    you: "あなた",
    leaveButton: "退室",
  },
  panel: {
    title: "eKYC認証",
    reset: "リセット",
    // ... all other panel keys
  },
};
```

## Locale Structure

```typescript
interface EkycLocale {
  preview: EkycPreviewLocale; // Preview component strings
  room: EkycRoomLocale; // Room component strings
  panel: EkycPanelLocale; // Action panel strings
}
```

### Preview Locale Keys

| Key           | English             | Vietnamese               |
| ------------- | ------------------- | ------------------------ |
| `title`       | Device Check        | Kiểm tra thiết bị        |
| `description` | Test your camera... | Kiểm tra camera...       |
| `joinButton`  | Join Session        | Tham gia phiên thẩm định |
| `joining`     | Connecting...       | Đang kết nối...          |
| `cameraLabel` | Camera              | Camera                   |
| `micLabel`    | Microphone          | Micro                    |
| `cameraOff`   | Camera is off       | Camera đã tắt            |

### Room Locale Keys

| Key           | English                       | Vietnamese                |
| ------------- | ----------------------------- | ------------------------- |
| `connecting`  | Connecting to meeting room... | Đang kết nối phòng họp... |
| `hostLabel`   | Appraiser (HOST)              | Thẩm định viên (HOST)     |
| `guestLabel`  | Customer (GUEST)              | Khách hàng (GUEST)        |
| `you`         | You                           | Bạn                       |
| `leaveButton` | Leave Room                    | Rời phòng                 |

### Panel Locale Keys

The panel has the most keys for document types, actions, and result labels. See the [source code](https://github.com/ermisnetwork/ermis-ekyc-sdk/blob/main/packages/react/src/locale/en.ts) for the complete list.
